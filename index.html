HTML & CSS

HTML est un langage utilisé pour créer la structure d'un site web. Il permet d'ajouter du texte, des images, des liens et des formulaires.

CSS est un langage utilisé pour styliser les pages HTML. Il permet d'ajouter des couleurs, des animations et un design moderne.
